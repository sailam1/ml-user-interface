Data preprocessing

  1. Features

    • these are the columns which are used to find relation between predicting columns and input data
    • make sure these are accurate as much as possible

  2. To predict

    • column which contains data to predict

  3. choose to replace missing values

    • the method choosen will replace the empty values with these values

  4. choose column number which has type "names"

    • to replace word type of data to an integer type data

Machine Learning

  1. Regression Problem

    • if the output/y column has continuous values
    • like numbers from 0 to infinity with float and integer type
    • Ex...205,208.46,4000,915.26,etc
    • choose regression problem

  2. Classification Problem

    • if the output/u column has discreate values
    • like only integer or float
    • EX...0,1,2,2,1,0,4,etc

  3. Clustering

    • this algorithms are used for grouping
    • ex..if mall has varity of customers and they want to group customers on their likes and dislikes .they your output will have two groups
